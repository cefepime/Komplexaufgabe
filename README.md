## Komplexaufgabe

Programmieraufgabe eines viertes Semester einer DHBW in **Informatik** 2024.

Module:
- Software Engineering I
- Software Engineering II

Erstes Modul behandelt SOLID zweites Modul OOP Patterns. Während der Vorlesungen bestehen immer wieder Team- und Einzelaufgaben, die in Moodle hochgeladen werden müssen. Diese kosten auch viel Zeit. Die Bewertung der Komplexaufgabe gibt eine gemeinsame Note für beide Module.

Falls jemand wieder meint duales Studieren sei easy... Ich möchte mich nicht beschweren aber wäre mal interessant wie sich das mit anderen, ähnlichen Studiengängen überschneidet.

Der Prof. setzt für die Aufgabe 20h als Einsatz an... löl. Wir haben aber einen Monat Zeit für den Upload.
